# Bhawan's Boutique Website

Premium custom tailoring boutique website for Bhawan Tayal.

## How to make this live on GitHub Pages

### Step 1 — Create a GitHub repository
1. Go to [github.com](https://github.com) and sign in
2. Click the **+** button → **New repository**
3. Name it `bhawans-boutique` (or any name you like)
4. Set it to **Public**
5. Click **Create repository**

### Step 2 — Upload these files
1. In your new repository, click **Add file** → **Upload files**
2. Upload ALL the files in this folder:
   - `index.html`
   - `assets/` folder (with both files inside it)
3. Click **Commit changes**

### Step 3 — Enable GitHub Pages
1. Go to your repository **Settings**
2. Click **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch, **/ (root)** folder
5. Click **Save**
6. Wait 1–2 minutes — your site will be live at:
   `https://YOUR-GITHUB-USERNAME.github.io/bhawans-boutique/`
